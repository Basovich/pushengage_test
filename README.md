# PushEngage Test Project

This is a simple Webpack project setup.

## Installation

Install the dependencies:

```bash
npm install
```

## Development

Start the development server with hot reload:

```bash
npm start
```

This will run the server at `http://localhost:8080`. The page will automatically reload if you change any source files (HTML, CSS, JS).

## Build

Build the project for production:

```bash
npm run build
```

The output files will be generated in the `dist` directory.

## Deployment

### Vercel

To deploy to Vercel:

1.  Install Vercel CLI: `npm i -g vercel`
2.  Run `vercel` in the project root.
3.  Follow the prompts. The `vercel.json` configuration will handle the build settings.
